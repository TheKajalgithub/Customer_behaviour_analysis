# 📊 Customer Behaviour Data Analysis Project

## 📌 Overview

This project focuses on analyzing customer behavior using a complete data analysis pipeline. It covers data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and dashboard visualization to generate meaningful business insights.

---

## 🎯 Objectives

* Understand customer purchasing patterns
* Analyze the impact of discounts on sales
* Identify trends across different customer segments
* Build an interactive dashboard for business decision-making

---

## 🛠️ Tech Stack

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **PostgreSQL** (SQL queries & data analysis)
* **Power BI** (Dashboard & visualization)

---

## 📂 Project Workflow

### 1️⃣ Data Loading (Python)

* Imported dataset using Pandas
* Performed initial inspection of data structure
* Checked for missing values and data types

---

### 2️⃣ Data Cleaning

* Handled missing/null values
* Removed duplicates
* Standardized column formats
* Converted data types where required

---

### 3️⃣ Exploratory Data Analysis (EDA)

* Analyzed key metrics like revenue, purchases, and customer distribution
* Visualized trends using graphs and charts
* Identified patterns and anomalies

---

### 4️⃣ SQL Analysis (PostgreSQL)

* Loaded cleaned data into PostgreSQL
* Wrote optimized SQL queries to extract insights:

  * Revenue by age group
  * Discount impact on purchases
  * Customer segmentation analysis
  * Purchase frequency trends

---

### 5️⃣ Dashboard Creation (Power BI)

* Built an interactive dashboard to visualize insights
* Included KPIs such as:

  * Total Revenue
  * Customer Segments
  * Discount Usage
  * Sales Trends
* Enabled filtering for better analysis

---

## 📊 Key Insights

* Certain age groups contribute more to total revenue
* Discounts significantly influence purchasing behavior
* Repeat customers generate higher overall value
* Seasonal or category-based trends observed
